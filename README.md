# crypto-hash-poc
hash collision PoC

## Repository clone
```bash
$ git clone https://github.com/megasia/crypto-hash-poc.git
```

## Create virtualenv and activate

```bash
$ virtualenv venv
$ source ./venv/bin/activate
$ pip install -r requirements.txt
```

## Run

Run tests to check
```bash
$ cd tests
$ python main_test.py
```
